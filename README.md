> [!IMPORTANT]
> **Arquitectura Optimizada**
>
> El núcleo de GUERRA BOT MD está diseñado para trabajar bajo una arquitectura híbrida de persistencia utilizando **MongoDB™** y almacenamiento local inteligente.
>
> El sistema implementa:
>
> - Cache de alto rendimiento
> - Carga dinámica de plugins
> - Workers independientes
> - Reconexión automática
> - Gestión avanzada de eventos
> - Sistema modular ESM

---

> [!TIP]
> **Compatibilidad Flexible**
>
> En ausencia de una instancia MongoDB configurada, el sistema puede trabajar bajo almacenamiento local sin detener el funcionamiento principal del bot.

---

> [!WARNING]
> **Producción**
>
> Para entornos con tráfico elevado se recomienda:
>
> - MongoDB Atlas
> - PM2
> - VPS Linux
> - NodeJS LTS v20+
>
> El uso de almacenamiento local en bots grandes puede generar degradación de rendimiento.

# 🚀 Instalación

## 📱 Termux

```bash
pkg update && pkg upgrade -y
pkg install git nodejs-lts ffmpeg imagemagick -y
```

---

## 📂 Clonar Repositorio

```bash
git clone https://github.com/TU-USUARIO/guerra-bot-md
cd guerra-bot-md
```

---

## 📦 Instalar Dependencias

```bash
npm install
```

---

## ▶️ Iniciar Bot

```bash
npm start
```

---

# ⚡ Producción VPS

Para máxima estabilidad:

```bash
npm install -g pm2
pm2 start index.js --name GUERRA-BOT
pm2 save
```
